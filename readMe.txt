Name: UCL Foosball 19
Project Description:
-A multi/single player soccer game modeled after the famous UEFA Champions League Football Tournament
-Manage Team, set Tactics, rotate Players
-Play Foosball matches and win game

How To Run
-Run Screens.py (if pygame and moviepy are sucessflly installed)
-Cut and Paste all images from image file into same directory as python programs

Installing Libraries
-Run Austin Schick's module manager
- Install moviePy and pygame
